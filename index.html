Tudo começa com Pixels

Para você, uma imagem é uma foto colorida. Para o computador, é uma matriz gigante de números (como uma planilha do Excel).

Cada "quadradinho" (pixel) tem um valor numérico que diz o quão brilhante ou colorido ele é. Tente diminuir a resolução ao lado para ver a grade se formar.
